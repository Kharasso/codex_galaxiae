\ AstroDex

ASTROCODEX MAPPA GALAXIAE

Project Team

Ruijun Liu, Huopu Zhang

Project Description

This project aims at creating a visualized CODEX for stars that are documented in the HYG Celestial Body Dataset. On one hand, it visualizes the distribution of non-spatial features of stars, which offers a way for the audience to explore stars by shared qualities. On the other hand, it also provides the user with the opportunity to explore these selected stars in space, showing them a spatial layout and also real telescopic snapshots taken from Aladdin Lite API.
